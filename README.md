# Chaos Hub 🌀

Welcome to **Chaos Hub**! This is an Android application built to provide a seamless mobile experience, featuring integrated web capabilities and custom native transitions.

## 🚀 Features
* **Web-to-Native Integration:** Built with a high-performance WebView engine.
* **Custom UI:** Includes custom CSS and JavaScript injections for a unique look.
* **Media Handling:** Support for file downloads and media viewing.
* **Fast Navigation:** Optimized with custom progress bars and smooth transitions.

## 📲 Installation
You can download the latest version of the app from the **Releases** section of this repository.

1. Download the `app-release.apk`.
2. Transfer it to your Android device.
3. Enable "Install from Unknown Sources" in your settings.
4. Install and enjoy!
